# 🎨 Aayush - Premium Web Design & Development Portfolio

A modern, fully-featured portfolio website showcasing web design and development expertise. Built with pure HTML5, CSS3, and vanilla JavaScript for optimal performance.

## ✨ Features

### Design
- ✅ **Modern Dark Theme** - Premium glassmorphism aesthetic
- ✅ **Fully Responsive** - Mobile, tablet, desktop optimized
- ✅ **Smooth Animations** - Scroll reveals, hover effects, transitions
- ✅ **Custom Cursor** - Interactive desktop cursor tracking
- ✅ **Glassmorphism** - Frosted glass effects with backdrop blur

### Functionality
- ✅ **Project Showcase** - 6 featured projects with tech stack
- ✅ **Skills Section** - Animated progress bars for technical skills
- ✅ **Testimonials** - Client reviews with 5-star ratings
- ✅ **Pricing Plans** - 3 flexible pricing tiers
- ✅ **Contact Form** - Professional contact form integration ready
- ✅ **FAQ Section** - Accordion-style Q&A
- ✅ **Mobile Menu** - Smooth overlay navigation
- ✅ **Scroll Progress** - Visual scroll progress indicator

### Performance
- ✅ **Fast Loading** - No frameworks, pure HTML/CSS/JS
- ✅ **SEO Optimized** - Structured markup, meta tags
- ✅ **Hardware Accelerated** - Smooth 60fps animations
- ✅ **Mobile First** - Touch-optimized interactions
- ✅ **Accessibility** - ARIA labels, semantic HTML
