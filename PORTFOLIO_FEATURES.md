# Aayush Premium Portfolio Website

## Overview
A modern, premium dark-themed portfolio website built with HTML5, CSS3, and vanilla JavaScript. Designed to showcase web design and development expertise with smooth animations, glassmorphism effects, and a professional layout.

---

## ✨ Key Features

### 🎨 Design & Theme
- **Modern Dark Luxury Theme**: Deep black background with purple/indigo accent colors
- **Glassmorphism Effects**: Frosted glass cards with backdrop blur
- **Smooth Gradients**: Premium linear and radial gradients throughout
- **Animated Ambient Background**: Floating orbs, beams, and grid patterns
- **Professional Typography**: Plus Jakarta Sans font for modern aesthetics

### 🚀 Performance & Optimization
- **Fully Responsive**: Mobile, tablet, and desktop optimized
- **SEO Optimized**: Structured semantic HTML, meta tags, schema markup
- **Fast Loading**: Optimized CSS and minimal JavaScript
- **Smooth Scrolling**: Premium scroll behavior with progress tracking
- **Hardware Accelerated**: CSS transforms for smooth animations

## 🚀 How to Use

1. **Open in Browser**: Simply open `index.html` in any modern browser
2. **Local Server**: Run `python3 -m http.server 8000` and visit `http://localhost:8000`
3. **Customize Content**: 
   - Replace portfolio project images and details
   - Update testimonials with real client reviews
   - Modify pricing plans to match your services
   - Add social media links
4. **Deploy**: Upload to any web hosting service

## 📝 Customization Guide

### Update Hero Section
```html
<!-- In HTML, find the hero section and update: -->
<h1>Your Title Here</h1>
<p>Your tagline here</p>
```

### Update Skills
```html
<!-- In CSS, modify the --skill-percent value: -->
<div class="skill-bar" style="--skill-percent: 95%;">
```

### Update Projects
```html
<!-- Replace image URLs and project details: -->
<img src="your-project-image.jpg" alt="Project name">
```

### Update Colors
```css
:root {
  --primary: #your-primary-color;
  --primary-2: #your-secondary-color;
}
```
